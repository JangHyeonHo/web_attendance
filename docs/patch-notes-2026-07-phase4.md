## D25. 사계절 테마 — 시스템 전역 설정(SYSTEM_ADMIN) + AUTO 계절 자동

- **결정**: `app_setting('UI_THEME')`(V11)에 `AUTO|SPRING|SUMMER|AUTUMN|WINTER`를 저장하고,
  `GET/PUT /api/v1/admin/ui-theme`(RoleInterceptor `/api/v1/admin/**` 기존 규칙 편승)로 W004에서 변경한다.
  **확정 테마는 navigation 응답(`theme`)에 동봉** — 프론트는 `<html data-theme>`에 반영만 하고,
  CSS 토큰(`[data-theme=...]`)이 전 화면 색을 바꾼다. AUTO는 서버 날짜의 계절로 해석(3–5월 봄 / 6–8월 여름 / 9–11월 가을 / 12–2월 겨울)되어 **프론트에는 AUTO가 절대 오지 않는다**.
- **이유**: 공개 화면(랜딩/로그인)도 테마가 적용되어야 하므로 별도 조회 API보다 화면 전개 응답 동봉이 자연스럽다(왕복 0회 추가). 요구가 "시스템 관리자가 설정"이므로 테넌트/사용자별이 아닌 시스템 전역 1값.
- **리스크/한계**: 계절 판정은 북반구 기준(서버 시간대). 남반구 고객은 고정 테마로 운용. 테넌트별 테마는 후속 후보.
- **폴백**: 설정 행 누락/미지 값은 AUTO로 폴백 — 테마가 기동·화면 전개를 막지 않는다.

## D26. 모달 일원화 — 등록 폼·파괴적 확인의 테이블 내 확장 행 폐지

- **결정**: 공통 `Modal`(PC 중앙 다이얼로그 / 모바일 바텀시트, ESC·배경 클릭 닫기, danger 변형)을 신설하고
  등록 폼(W004 텍스트, W007 테넌트, W009 멤버, W013 공휴일), 파괴적 확인(정지/삭제/강등/비활성/동기화),
  발송 전 이메일 재확인, 출결 스탬프 확인을 전부 모달로 이전. 목록 상단은 `.toolbar`(제목+주 버튼)로 통일.
  **행 아래 인라인 표시는 에러만 유지**(맥락 보존).
- **이유**: 테이블 안에 `colSpan` 확장 행으로 패널을 심는 기존 방식은 모바일에서 가로 스크롤과 겹쳐 조작 불능에 가까웠다. 오송신 방지 재확인(W009)의 "취소 시 폼 값 유지" 계약은 모달 전환 후에도 유지(확인 취소 → 등록 모달 복귀).
- **기대값**: 스크린 리더 계약 유지(dialog/alertdialog + aria-modal), 바텀시트는 `env(safe-area-inset-bottom)` 반영.

## D27. 디자인 토큰 재작성 + 테이블 통일 + 모바일 1급 시나리오

- **결정**: `app.css`를 토큰 기반으로 전면 재작성(외부 리소스 0 유지 — 시스템 폰트·이모지만).
  - 테이블: `.table-wrap`(카드형 래퍼 + 가로 스크롤) + 통일 패딩/행 높이/헤더, `tabular-nums`.
    긴 텍스트 열만 `td.wrap`으로 줄바꿈 허용(기본은 nowrap — 모바일 스크롤 유지).
  - 출결(W005): 스탬프 4버튼을 대형 터치 그리드(PC 4열/모바일 2×2, 76px+)로. 터치 단말은 모든 버튼 44px+.
  - 헤더: sticky + blur, 내비 필(현재 화면 `aria-current` 강조), 모바일은 한 줄 가로 스크롤.
- **이유**: "출결은 모바일로 찍는다" — 스탬프 동선(로그인→출근→확인)을 엄지 조작만으로 완결.
  열 폭 들쭉날쭉의 원인은 화면마다 제각각인 테이블 마크업이 아니라 스타일 부재였으므로 클래스 계약(`detail-table`)은 유지한 채 스타일만 통일(화면 수정 최소화).

### 관찰(비수정 — 후속 후보)

- **statusLabel의 언어**: 출결 상태 라벨(`messages` 해석)은 세션 lang이 아닌 요청 로케일(Accept-Language)을 따른다.
  브라우저 언어=앱 언어인 일반 사용자는 차이가 없지만, 앱에서 언어만 바꾼 경우 상태 라벨이 브라우저 언어로 남는다.
  Phase 4 이전부터의 동작(회귀 아님)이라 이번 범위에서 수정하지 않음 — LocaleResolver를 세션 lang과 동기화하는 개선이 후속 후보.
