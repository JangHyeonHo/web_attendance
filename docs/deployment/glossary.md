# 용어집 (구축 문서용)

구축 문서에서 처음 나오는 순서대로. "정확한 정의"보다 "이 문서를 따라가는 데 필요한 만큼"만 설명한다.

## 계정·OCI

| 용어 | 뜻 |
|---|---|
| **테넌시(Tenancy)** | 오라클 클라우드에서 "내 계정 전체"를 부르는 말. 가입 때 정하는 클라우드 계정 이름 = 테넌시 이름 |
| **리전(Region)** | 데이터센터가 있는 지역(서울/춘천/도쿄/오사카…). 서버는 리전 안에 만들어진다 |
| **홈 리전(Home Region)** | 가입 시 정하는 기본 리전. **변경 불가**이며 무료 ARM 서버는 여기서만 생성 가능 |
| **AD(Availability Domain)** | 리전 안의 독립된 데이터센터 묶음. 무료 서버 재고가 AD별로 달라서 생성 실패 시 바꿔 시도한다 |
| **인스턴스(Instance)** | 가상 서버 1대. "서버를 만든다" = "인스턴스를 생성한다" |
| **셰이프(Shape)** | 서버의 하드웨어 구성 메뉴명(CPU 종류·개수·메모리). A1.Flex = ARM CPU의 무료 대상 셰이프 |
| **OCPU** | 오라클식 CPU 단위. 대략 "코어 수"로 읽으면 된다 |
| **Always Free** | 기간 제한 없이 무료인 리소스 범위(ARM 합계 4 OCPU/24GB 등). 30일 체험 크레딧과는 별개 |
| **VCN(Virtual Cloud Network)** | 내 서버들이 소속되는 가상 네트워크. 인스턴스 생성 마법사가 자동으로 만들어 준다 |
| **서브넷(Subnet)** | VCN 안의 구획. 우리 구성에서는 자동 생성된 것 하나만 쓴다 |
| **보안 목록(Security List)** | 클라우드 쪽 방화벽 규칙(어느 포트를 열지). 서버 안 iptables와 **별개** — 둘 다 열어야 통신된다 |
| **인그레스(Ingress)** | 밖→서버 방향 통신 규칙. 반대말은 이그레스(서버→밖) |
| **예약 공인 IP(Reserved Public IP)** | 인스턴스를 지워도 유지되는 고정 공인 IP. 기본(Ephemeral)은 인스턴스와 함께 사라진다 |

## 접속·서버 안

| 용어 | 뜻 |
|---|---|
| **SSH** | 원격 서버에 터미널로 접속하는 표준 방법. `ssh` 명령 |
| **SSH 키(공개키/개인키)** | 비밀번호 대신 쓰는 파일 열쇠 한 쌍. 공개키(.pub)는 서버에 등록, 개인키는 내 PC에 보관(유출 금지) |
| **iptables** | 리눅스 OS 안의 방화벽. OCI 우분투 이미지는 여기에 거부 규칙이 미리 들어 있다(단골 함정) |
| **systemd / 서비스** | 리눅스의 프로그램 상주 관리자. "서비스로 등록" = 부팅 시 자동 시작 + 죽으면 재시작 |
| **nano** | 터미널용 간단 텍스트 편집기. `sudo nano <파일경로>`로 열고, Ctrl+O(저장)·Enter·Ctrl+X(닫기) |
| **환경변수** | 프로그램에 설정값을 주입하는 통로. 우리는 `/etc/attendance/attendance.env` 파일에 모아 둔다 |

## 웹·도메인

| 용어 | 뜻 |
|---|---|
| **DNS** | 도메인 이름 → IP 주소 변환 체계 |
| **A 레코드** | "이 이름은 이 IP다"라는 DNS 등록 항목. `app.<도메인> → 서버 IP` 로 1건 등록한다 |
| **와일드카드 레코드/인증서** | `*.app.<도메인>`처럼 하위 이름 전부를 한 번에 커버하는 것. 테넌트 서브도메인(11번 문서)에서 필요 |
| **TLS/HTTPS 인증서** | 브라우저 자물쇠. Let's Encrypt에서 무료 발급(certbot 도구) |
| **리버스 프록시** | 브라우저의 요청을 받아 뒤의 백엔드로 전달하는 앞단 서버 — 우리 구성의 nginx 역할 |
| **레지스트라** | 도메인을 구매·관리하는 업체(Cloudflare, 가비아, お名前.com 등) |

## 앱 쪽

| 용어 | 뜻 |
|---|---|
| **프로파일(dev/prod)** | Spring 설정 묶음 전환 스위치. 공개 서버는 항상 `prod`([01](01-architecture.md)) |
| **Flyway / 마이그레이션** | DB 스키마를 버전 파일(V1, V2, …)로 관리하는 도구. 앱이 기동하며 자동 적용 — 수동 DDL 불요 |
| **jar** | 자바 앱을 파일 하나로 묶은 실행 파일. `java -jar app.jar`로 실행 |
| **테넌시(우리 서비스의)** | 우리 SaaS의 "고객사" 단위. OCI의 테넌시와는 무관한 동음이의어이니 혼동 주의 |
