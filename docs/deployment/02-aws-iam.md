# 02. AWS 계정 준비 — 루트 보호와 IAM 사용자

> 이전: [01. 전체 구성](01-architecture.md) · 다음: [03. EC2 인스턴스 만들기](03-aws-ec2.md) / [인덱스](README.md)
> 모르는 용어는 [용어집](glossary.md).

전제: **AWS 루트 계정(가입 시 이메일로 로그인하는 계정)은 이미 있다.**
이 문서가 끝나면: 루트는 봉인되고, 일상 작업용 관리자 IAM 사용자로 콘솔을 쓰는 상태가 된다.

루트 계정을 평소에 쓰지 않는 이유: 루트는 결제·계정 폐쇄까지 가능한 전능 계정이라
유출 시 복구 수단이 없다. 일상 작업은 권한을 부여받은 IAM 사용자로 하고,
루트는 "IAM이 잠겼을 때의 비상 열쇠"로만 남긴다.

## 2-1. 루트 계정 보호 (MFA)

1. https://console.aws.amazon.com 에 **루트 사용자**(이메일)로 로그인
2. 우상단 계정 이름 클릭 → **보안 자격 증명(Security credentials)**
3. "멀티 팩터 인증(MFA)" → **MFA 디바이스 할당(Assign MFA device)**
   - 유형: **인증 관리자 앱(Authenticator app)** 선택
   - 휴대폰의 인증 앱(Google Authenticator, Microsoft Authenticator 등)으로 QR 스캔
   - 앱에 뜨는 6자리 코드를 연속 2회 입력 → 등록 완료
4. 이후 루트 로그인에는 비밀번호+앱 코드가 함께 필요해진다.
   ★휴대폰을 바꿀 때 인증 앱 이전을 잊지 말 것 — 루트 MFA 분실은 복구가 매우 번거롭다★

이 화면에서 **계정 ID(12자리 숫자)** 도 확인해 메모해 둔다(우상단 메뉴에도 표시됨).

## 2-2. 관리자 IAM 사용자 만들기

1. 콘솔 상단 **검색창에 "IAM"** 입력 → IAM 서비스로 이동
2. 왼쪽 메뉴 **사용자(Users)** → **사용자 생성(Create user)**
3. 단계별 입력:
   - 사용자 이름: 예) `admin-<본인이름>`
   - **"AWS Management Console에 대한 사용자 액세스 권한 제공" 체크**
     → "IAM 사용자를 생성하고 싶음" 선택
     → 콘솔 비밀번호: "사용자 지정 비밀번호"로 직접 설정
     → "다음 로그인 시 새 비밀번호 생성 필요"는 본인용이므로 **체크 해제**해도 된다
4. 권한 설정: **"직접 정책 연결(Attach policies directly)"** →
   검색창에 `AdministratorAccess` → 같은 이름의 정책 체크 → 다음 → **사용자 생성**
5. 생성 완료 화면의 **콘솔 로그인 URL**을 북마크:
   `https://<계정ID>.signin.aws.amazon.com/console`

## 2-3. IAM 사용자로 다시 로그인 + MFA

1. 루트를 로그아웃하고, 위 로그인 URL에서 **IAM 사용자**로 로그인
   (계정 ID / 사용자 이름 / 비밀번호)
2. 이 사용자에게도 MFA를 건다: IAM → 사용자 → 본인 클릭 → 보안 자격 증명 탭 →
   MFA 디바이스 할당(방법은 2-1과 동일. 인증 앱에는 루트와 별개 항목으로 등록된다)
3. **이후 모든 작업(03 이후 문서 전부)은 이 IAM 사용자로 한다.** 루트는 로그아웃.

액세스 키(프로그램용 키)는 **만들지 않는다** — 이 구축은 콘솔+SSH만으로 끝나며,
쓰지 않는 키는 유출 통로일 뿐이다. 나중에 CLI가 필요해지는 시점에 그때 만든다.

## 2-4. 요금 알림 걸어두기 (사고 방지)

유료 계정이므로 "모르는 새 과금"을 알림으로 막는다:

1. 콘솔 검색창 "Budgets" → **예산 생성(Create budget)**
2. 템플릿: **월별 비용 예산(Monthly cost budget)** → 금액: 예) $10
   (03에서 만들 서버의 예상 월액보다 약간 위로 — 초과 조짐이 보이면 메일이 온다)
3. 알림 이메일 입력 → 생성

## 완료 확인

- [ ] 루트에 MFA 등록됨
- [ ] IAM 사용자(AdministratorAccess)로 콘솔 로그인 성공, MFA 등록됨
- [ ] 콘솔 로그인 URL(계정 ID 포함) 북마크
- [ ] 월별 예산 알림 1건 생성
- [ ] 루트는 로그아웃 상태(이후 사용 안 함)
